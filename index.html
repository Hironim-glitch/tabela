Walidacja strony HTML to proces sprawdzania, czy kod HTML strony internetowej jest zgodny ze standardami i specyfikacjami ustalonymi przez organizacje, takie jak W3C (World Wide Web Consortium). Celem walidacji jest zapewnienie, że strona działa poprawnie we wszystkich przeglądarkach, jest dostępna dla użytkowników, w tym osób z niepełnosprawnościami, oraz że jest łatwa do indeksowania przez wyszukiwarki. Walidacja może wykrywać błędy składniowe, niezgodności z normami i inne problemy, które mogą wpływać na wydajność i użyteczność strony.

Walidator to narzędzie lub program, który sprawdza, czy dany dokument (np. kod HTML, CSS czy XML) jest zgodny z określonymi standardami i specyfikacjami. Walidatory analizują kod, wykrywają błędy, ostrzegają o niezgodnościach oraz sugerują poprawki. Dzięki temu pomagają zapewnić, że strony internetowe są poprawne, wydajne i dostępne dla użytkowników. Przykładami walidatorów są W3C Markup Validation Service dla HTML oraz CSS Validation Service dla arkuszy stylów.
